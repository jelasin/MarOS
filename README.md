## 项目简介

MarOS是一个基于x86-64汇编语言的多处理器多线程操作系统项目。本项目采用QEMU模拟器运行，使用NASM作为编译工具，旨在帮助读者掌握操作系统基本知识，为后续编写功能更为完善的操作系统奠定基础。
